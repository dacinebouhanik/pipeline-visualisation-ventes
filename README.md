# Pipeline de consolidation et visualisation de données (Python)

##  Description

Ce projet implémente un mini pipeline de traitement de données permettant de consolider des ventes mensuelles issues de plusieurs fichiers Excel (Octobre, Novembre, Décembre).

Les données sont extraites, agrégées par article, puis exportées dans un nouveau fichier Excel incluant un graphique généré automatiquement.

L'objectif est d'automatiser un processus de reporting et de démontrer la manipulation programmatique de données via Python.

---

##  Fonctionnalités

- Lecture de plusieurs fichiers Excel
- Agrégation des ventes par article
- Structuration des données via dictionnaire Python
- Génération automatique d’un fichier consolidé
- Création dynamique d’un graphique en barres
- Automatisation complète du reporting trimestriel

---

##  Technologies utilisées

- Python
- OpenPyXL
- Manipulation et agrégation de données
- Génération programmatique de visualisations Excel
